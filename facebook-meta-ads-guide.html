Product Catalog Setup Guide for MyPupCentral

üìã Overview

This guide provides comprehensive recommendations for setting up a Product Catalog in Meta Ads Manager to effectively showcase your puppy listings in Facebook and Instagram ads.

üîë Key Understanding:

The Pixel alone is not sufficient for product catalogs. The Pixel tracks user behavior and events (page views, purchases, etc.), while a Product Feed provides the actual product data (puppy listings, prices, images, descriptions). You need BOTH working together for dynamic product ads.

üéØ Recommended Method: Product Feed via URL

For your dynamic pet marketplace, the best approach is to use a Product Feed (CSV/XML) hosted on your website.

Why Product Feed is the Best Choice:

‚úÖ Automatic Updates

Feed updates automatically when listings change - no manual work required

‚úÖ Dynamic Ads

Show specific puppies to users who viewed similar breeds

‚úÖ Retargeting

Show puppies users viewed but didn't inquire about

‚úÖ Scale Easily

Handle hundreds or thousands of listings without effort

‚úÖ Better ROI

More relevant ads lead to better performance and lower costs

‚úÖ Real-Time Inventory

Only show available puppies, hide adopted ones automatically

üìù Step-by-Step Setup Process

1 Create a Product Feed File

You need to generate a CSV or XML file with your puppy listings. This feed should be automatically generated from your database.

Required Fields (Must Include):

Field Name Description Example
id Unique identifier for each puppy 12345
title Puppy name/breed "Golden Retriever Puppy - Male"
description Detailed puppy information "Beautiful golden retriever puppy, 8 weeks old..."
availability in stock / out of stock "in stock" or "out of stock"
condition Product condition "new" (always "new" for puppies)
price Price with currency code "1200 USD"
link Direct URL to puppy listing page "https://mypupcentral.com/puppy/12345"
image_link URL to main puppy photo "https://mypupcentral.com/uploads/puppies/image.jpg"
brand Breeder name or site name "Happy Tails Breeders" or "MyPupCentral"

Recommended Additional Fields (For Better Performance):

  • additional_image_link - URLs to more puppy photos (up to 10)
  • google_product_category - Use: "Animals & Pet Supplies > Live Animals"
  • product_type - Breed category (e.g., "Dogs > Golden Retriever")
  • sale_price - Discounted price if offering promotions
  • custom_label_0 - Use for "Breed Type" (Purebred, Mixed, Designer)
  • custom_label_1 - Use for "Size" (Toy, Small, Medium, Large, Giant)
  • custom_label_2 - Use for "Age Range" or "Sex" (Male/Female)
  • custom_label_3 - Use for "Price Range" ($500-$1000, $1000-$2000, etc.)
  • custom_label_4 - Use for "Location" (State or Region)

2 Host the Feed on Your Website

The feed should be accessible via a public URL on your website.

Recommended Feed URL Structure:

Primary Feed: https://mypupcentral.com/feed/facebook-catalog.xml

Alternative Formats:

  • CSV Format: https://mypupcentral.com/feed/facebook-catalog.csv
  • TSV Format: https://mypupcentral.com/feed/facebook-catalog.tsv

Feed Requirements:

  • ‚úÖ Publicly accessible (no login required)
  • ‚úÖ Updated automatically when listings change
  • ‚úÖ Properly formatted according to Meta's specifications
  • ‚úÖ Include only available (not adopted) puppies
  • ‚úÖ Maximum file size: 100 MB (compressed) or 1 GB (uncompressed)
  • ‚úÖ Response time: Under 10 seconds

3 Set Up Catalog in Meta Commerce Manager

Once your feed is ready, follow these steps in Meta:

  1. Go to Meta Commerce Manager: commerce.facebook.com
  2. Click "Create Catalog"
  3. Choose "E-commerce" as the catalog type
  4. Select "Product Feed" as the connection method
  5. Choose "Scheduled Feed"
  6. Enter your feed URL: https://mypupcentral.com/feed/facebook-catalog.xml
  7. Set update frequency:
    • Recommended: Daily (for active inventory)
    • Alternative: Hourly (if inventory changes very frequently)
    • Minimum: Weekly
  8. Click "Upload" to test the feed
  9. Review any errors or warnings
  10. Once validated, click "Finish Setup"

4 Connect Pixel to Catalog

Link your existing Facebook Pixel to the catalog for tracking and optimization:

  1. In Commerce Manager, go to your catalog
  2. Click "Settings" ‚Üí "Data Sources"
  3. Click "Add Data Source"
  4. Select "Pixel"
  5. Choose your existing Pixel from the dropdown
  6. Click "Add"
  7. Enable "Automatic Item Updates" (recommended)

5 Set Up Dynamic Ads Events

Add these Pixel events to your website for better tracking:

Required Events:

  • ViewContent - When someone views a puppy listing page
  • AddToCart - When someone adds to favorites or initiates inquiry
  • InitiateCheckout - When someone starts the contact/application process
  • Purchase - When deposit is made or adoption is confirmed
  • Search - When someone searches for specific breeds

Important: Each event should include the product ID (content_ids) matching your feed's id field.

üíª Technical Implementation for MyPupCentral

Recommended Architecture:

Create a new controller endpoint in your CodeIgniter application:

Controller Structure:

File: application/controllers/Feed.php

Method: facebook_catalog()

URL: https://mypupcentral.com/feed/facebook-catalog

Feed Generation Logic:

  1. Query database for active puppy listings:
    • WHERE reviewed = 1 (approved ads)
    • AND is_sold = 0 (not adopted)
    • AND status = 'active' (active listings)
  2. For each puppy, gather:
    • Basic info (ID, title, breed, description)
    • Price information
    • Primary and additional images
    • Breeder information
    • Custom labels (size, age, etc.)
  3. Format as XML according to Meta specifications
  4. Set proper headers: Content-Type: application/xml
  5. Cache for 1 hour to improve performance
  6. Output the XML feed

Database Fields Mapping:

Meta Feed Field Your Database Field Notes
id posts.id Unique post ID
title posts.title or combine breed + name E.g., "Golden Retriever - Max"
description posts.description Strip HTML tags, limit 5000 chars
link base_url + '/puppy/' + posts.id Direct link to listing
image_link posts.featured_image Full URL to image
additional_image_link posts_pictures.picture_name Up to 10 additional images
price posts.price + ' USD' Format: "1500.00 USD"
availability is_sold ? 'out of stock' : 'in stock' Based on adoption status
brand user_accounts.name Breeder name
product_type breeds.breed_name From breeds table
custom_label_0 breed type or category Purebred, Mixed, Designer
custom_label_1 posts.puppy_size Toy, Small, Medium, Large
custom_label_2 posts.puppy_sex Male or Female

‚ö†Ô∏è Important Considerations

üö® Meta's Animal Advertising Policies

Critical: Facebook and Instagram have specific restrictions on live animal sales. Before implementing:

  • Review Meta's Animal Sales Policy
  • Live animal sales are generally prohibited in ads
  • Recommended approach: Frame your service as:
    • "Puppy Adoption Services"
    • "Breeder Directory & Connection"
    • "Pet Adoption Matching Service"
  • Focus on the service (connecting families with breeders) rather than direct sales
  • Consider working with Meta's support team to get pre-approval

üìã Compliance Checklist:

  • ‚úÖ Ensure all breeders are verified and legitimate
  • ‚úÖ Include health certifications in descriptions
  • ‚úÖ Clearly state breeding standards and practices
  • ‚úÖ Comply with local and state animal welfare laws
  • ‚úÖ Provide transparent pricing (no hidden fees)
  • ‚úÖ Include return/refund policies
  • ‚úÖ Verify age of puppies (typically 8+ weeks)

üîí Data Privacy & Security:

  • Ensure feed URL is using HTTPS (SSL certificate)
  • Don't include sensitive breeder information in public feed
  • Consider rate limiting to prevent feed scraping
  • Log feed access for monitoring purposes

üéØ Optimization Best Practices:

  • Image Quality: Use high-resolution images (minimum 1024x1024 pixels)
  • Titles: Include breed, age, sex, and key features
  • Descriptions: Be detailed but concise (500-1000 characters optimal)
  • Pricing: Keep prices competitive and up-to-date
  • Availability: Update immediately when puppies are adopted
  • Custom Labels: Use all 5 for better ad targeting

üß™ Testing & Validation

Before Going Live:

  1. Test Feed URL:
    • Access the feed URL in a browser
    • Verify it loads without errors
    • Check all required fields are present
    • Validate XML/CSV format is correct
  2. Use Meta's Diagnostic Tools:
    • Go to Commerce Manager ‚Üí Catalog ‚Üí Diagnostics
    • Review any errors or warnings
    • Fix issues before enabling ads
  3. Test with Small Sample:
    • Start with 10-20 listings
    • Verify they appear correctly in catalog
    • Create a test ad to preview
  4. Monitor Initial Upload:
    • Check upload status in Commerce Manager
    • Review item status (Active, Pending, Rejected)
    • Address any rejected items

‚úÖ Success Indicators:

  • Feed uploads successfully without errors
  • Products appear in catalog with correct information
  • Images load properly
  • Prices and availability are accurate
  • Links redirect to correct listing pages
  • Pixel events are firing correctly

üé® Creating Dynamic Ads Campaigns

Once Catalog is Set Up:

  1. Create New Campaign:
    • Go to Ads Manager
    • Click "Create" ‚Üí Choose "Sales" or "Traffic" objective
    • Enable "Advantage+ Catalog Ads"
  2. Ad Set Configuration:
    • Audience: Target people interested in dogs, pet adoption, specific breeds
    • Placements: Facebook Feed, Instagram Feed, Stories recommended
    • Product Set: Create sets for different categories:
      • By breed type (Small, Medium, Large)
      • By price range
      • By location
      • New listings
  3. Ad Creative:
    • Use Single Image or Carousel format
    • Create compelling ad copy
    • Use strong call-to-action: "Learn More", "Contact Us", "Apply Now"
  4. Retargeting Campaigns:
    • Create custom audience of people who viewed puppies but didn't inquire
    • Show them the same or similar puppies
    • Add urgency: "Still Available", "Price Drop", etc.
